## 2.异或(^)交换a,b的值
```
 // 交换a,b 的值
    a := 2
    b := 3

// 注意： (b & 1) == 1 可判断b是奇数
    if ((b & 1) == 1) {
        fmt.Println("转换前b是奇数, b=", b)
    }
    // 交换a,b
    a = a ^ b
    b = a ^ b
    a = a ^ b
    fmt.Printf("a=%d\n, b=%d\n\n", a, b)
    if ((a & 1) == 1){
        fmt.Println("转换后a是奇数, a=", a)
    }
```

```
输出:
转换前b是奇数, b= 3
a=3
, b=2
转换后a是奇数, a= 3
```
